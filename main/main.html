What is CS Hub?

We are a community that offers help in Java, JavaScript, C, C++, Python2 and Python3. This server/community will help if you ask questions and not only asking questions this community also gives you the opportunity to participate and develop for the community.

Need help with anything that is CS(Computer Science) related? Please join the server and ask for help and our helpers will try helping when they can.

Do you also want daily if not weekly challenges? Again, join the server and react for the challenger role in the *#verify* channel. We give competitive challenges every time and we have rewards for getting some challenges right. So join up and help our community grow!
